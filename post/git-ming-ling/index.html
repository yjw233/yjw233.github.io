git命令

· 2022-05-23 ·
  1. 查看远程仓库
    $ git remote -v
  2. 从远程获取最新版本到本地
    $ git fetch origin master
  3. 比较本地的仓库和远程参考的区别
    $ git log -p master.. origin/master
  4. 把远程下载下来的代码合并到本地仓库，远程的和本地的合并
    $ git merge origin/master

查看修改的状态
git status
指定commit包含的文件
git add
完成一次本地commit
git commit -a -m ""

将本地内容同步到云端
git push origin
